Chess Position Recognition Using Computer Vision

An open-source tool to convert chess board images to FEN notation

About the Project

ChessOCR is a computer vision project that recognizes chess positions from images and converts them to FEN (Forsyth-Edwards Notation), allowing for easy analysis and sharing of chess positions.

Whether you're analyzing a position from a book, screenshot, or photo of a physical board, ChessOCR aims to make the process seamless.

Project Goals

  • Accurately detect chess boards in various images
  • Identify pieces regardless of the board style or angle
  • Convert detected positions to standard FEN notation
  • Provide an easy-to-use interface for chess players
  • Support both web and mobile platforms
  • Maintain high accuracy across different lighting conditions

Features

Board Detection

Automatically locates and extracts the chess board from images

Piece Recognition

Identifies chess pieces using machine learning algorithms

FEN Notation

Converts the visual position to standard chess notation

Position Validation

Checks if the detected position is legal according to chess rules
